# Grocery Sales Analysis | Power BI & Python

## Project Overview
This project presents an end-to-end grocery sales analysis workflow using Power BI and Python to analyze sales performance, customer behavior, product trends, and operational efficiency.

The project combines Exploratory Data Analysis (EDA) using Python with interactive dashboard development in Power BI to generate actionable business insights and support data-driven decision-making.

---

## Business Problem
Retail businesses often struggle to:
- Monitor overall sales performance
- Identify profitable products and categories
- Track customer purchasing behavior
- Analyze delivery efficiency
- Understand profit and loss trends
- Detect data quality issues affecting reporting accuracy

This project helps transform raw grocery sales data into meaningful insights through data cleaning, EDA, KPI tracking, and interactive visualizations.

---

## Tools & Technologies
- Power BI
- Power Query
- DAX
- Python
- Pandas
- YData Profiling

---

## Python Contribution
Python was used during the Exploratory Data Analysis (EDA) phase to:
- Generate automated profiling reports using YData Profiling
- Identify potential data quality issues before data modeling

After EDA, the main cleaning, transformation, and modeling processes were completed using Power Query and Power BI to simulate a real-world business workflow.
