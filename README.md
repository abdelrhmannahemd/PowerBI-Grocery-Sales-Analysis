# Grocery Sales Analysis | Power BI

This project presents an end-to-end data analysis workflow using Power BI and Python.

## Project Overview
The project focuses on analyzing grocery sales data to understand sales performance, customer behavior, and employee contribution.

## Tools & Technologies
- Power BI
- Python
- YData Profiling
- Power Query
- DAX

## Project Workflow
1. Exploratory Data Analysis (EDA) using Python and YData Profiling
2. Data cleaning and validation
3. Calendar table handling using DAX
4. Building interactive dashboards in Power BI

## Key Dashboards
- Sales Overview
- Customers & Employees Analysis
